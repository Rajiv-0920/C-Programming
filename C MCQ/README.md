### 1. How many times abc print.

```c
#include <stdio.h>

int main(){

    for(int i = -1; i <= 10; i++){
        if(i < 5)
            continue;
        else
            break;
        printf("ABC");
    }

    return 0;
}
```

1. Infinite times
2. 11 times
3. **_0 times_**
4. 10 times
