### 30. Write a c program to check whether a given number is an Armstrong number or not.

**Example: 153 is an Armstrong number**

153 = (1 \* 1 \*1) + (5 \* 5 \* 5) + (3 \* 3 \* 3)

where:  
(1 \* 1 \* 1)=1  
(5 \* 5 \* 5)=125  
(3 \*3 \* 3)=27

So:  
1 + 125 + 27 = 153

153 is an Armstrong number
